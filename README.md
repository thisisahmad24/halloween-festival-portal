# 🎃 Halloween Festival Portal

An interactive **Halloween-themed web experience** built using **PHP, HTML, CSS, and JavaScript**.  
Visitors enter their names and get a spooky surprise — either a **💀 Trick** or a **🎁 Treat**, with their fate recorded in the backend.


## 🕸️ Features
- 🎭 Trick or Treat randomizer using PHP
- 💾 Saves results to a `results.txt` file
- 🌗 Dark/Light mode toggle
- 🕷️ Modular PHP structure with includes
- 👻 Floating ghost animation and spooky background
- 🔊 Sound effects for Trick and Treat


## ⚙️ How to Run
1. Install **XAMPP** and start **Apache**  
2. Copy the folder `halloween_festival` into C:\xampp\htdocs\
3. Open your browser and visit http://localhost/halloween_festival/
